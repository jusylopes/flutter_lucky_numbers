### 🔧 Instalação

1. Clone este repositório:
     
   ```bash
   git clone https://github.com/jusylopes/flutter_lucky_numbers.git
   cd lucky_numbers_app
   ```
3. Instale as dependências:
   ```bash
   flutter pub get
   ```
4. Execute no emulador ou dispositivo:
   ```bash
   flutter run
   ```
